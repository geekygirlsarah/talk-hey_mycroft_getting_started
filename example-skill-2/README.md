 Hashtag Reader
This will read up to 5 latest tweets from Twitter that match a hashtag

## About 
This will log into Twitter, pull down the latest 5 tweets that match a hashtag, then parse them out and read them out loud.

## Examples 
* "read hashtag"
* "read hashtags"
